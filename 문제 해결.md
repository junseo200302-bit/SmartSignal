# SmartSignal Troubleshooting

## Problem 1: Streamlit Dependency Error

### Issue
The application failed to run because required Python packages were missing.

### Solution
Installed the missing packages and created requirements.txt so the deployment environment can automatically install dependencies.

---

## Problem 2: Traffic Simulation Speed

### Issue
Vehicle movement and signal transitions were too slow during demonstrations.

### Solution
Adjusted animation speed and optimized the simulation timing to provide a smoother live demo experience.

---

## Problem 3: Unbalanced Signal Allocation

### Issue
Equal signal allocation caused unrealistic waiting times under heavy traffic.

### Solution
Implemented weighted demand scoring using vehicle count, pedestrian count, and emergency vehicle priority to dynamically allocate green-light duration.
